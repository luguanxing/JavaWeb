# JavaWeb项目

## 项目01-JavaWeb网上商城
```
而网上商城共分两个部分，
一部分是面向用户的部分，包括：顾客在线注册、购物、提交订单、付款等操作；
另外一部分是商城管理部分，这部分的内容包括：产品的添加、删除、查询、订单的管理、操作员的管理、注册用户的管理等。
```
