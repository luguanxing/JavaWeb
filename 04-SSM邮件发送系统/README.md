# SSM邮件发送系统

## 需求分析

```
使用SSM+Shiro+EasyUI实现一个邮件发送系统
能够手动发送邮件
能够增删改查邮件源
能够增删改查发送目标
能够根据计划自动发送邮件
能够增删改查计划
```
